Your new assignment has come in.

Last week the CEO of Eutropia LLC was murdered. This comes just days after the announcement of their new flagship product Zaira which threatens to unseat The Anastasia Corporation as the dominant force in social media.  

The grapevine says the murder was a hit organized by Anastasia executive leadership. We need you to prove it.

We've intercepted several batches of files from their internal servers, nevermind how we got them.Most of these files are encoded. You should have all the tools you need to decrypt and investigate. 

I know this isn't your usual field, frankly you just happen to be the only person with high enough clearance and enough free time to take on a task like this. I'll explain the basic tools.

Keygen:
The keygen program is a highly sophisticated tool that runs a vast array of encryption detecting algorithms. If you find two files that seem to share the same content, one encrypted and one decrypted, the keygen program will generate a key file which you can use to decode files with that same encryption.

Decoder:
Using the keys generated by keygen, the decoder program will unscramble the contents of any file encoded with the appropriate key.

Looking Glass:
We need you to solve certain mysteries for us. Using the looking glass program you will submit your solutions to each question. The program is smart enough to know if your solution is incorrect, if you're close, or if you're wrong.

We've provided you with a folder of reference material relevant to the case. Hopefully it will prove useful.

That's about it. Solve all of the mysteries and you can return to your regular duties.

For some reason we are unable to remove the minefield game from your system. Remember that your progress will be monitored.
